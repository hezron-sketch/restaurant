### LocalhostEats
#### A web app opening up market for the coastal restaurants, made with the `MERN Tech Stack` i.e Mongodb, Expressjs, React and Nodejs.
#### To run it locally on your computer:
#### Git clone the repo
- `git clone https://github.com/hezron-sketch/restaurant.git`
#### Install dependancies
- `cd backend && npm install`
- `cd frontend && npm install`
#### Create a `.env` file in the backend folder and add your local `MONGO_URI` and `JWT_SECRET`
#### In the project folder goto the backend folder and run the server by: `cd backend && npm run dev`
#### Finally go into the frontend folder and start the fronted application by: `cd frontend && npm start`

## NOTE:
- Assumptions are that you have mongodb running on your machine
- If you encounter any problems remember to let `Localhost Developers` know.
